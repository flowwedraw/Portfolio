Solana Sniper Bot + SaaS

Запущен

High-frequency trading bot · Rust · Solana · Telegram · PostgreSQL

Автономный торговый бот для покупки токенов на Solana по сигналам из Telegram-каналов. Система проходит полный цикл без участия пользователя: мониторинг, извлечение контракта, сборка транзакции, покупка, управление TP/SL/timeout и продажа.

  • Оптимизация задержки: keepalive-соединения, предсборка транзакций, прогрев подписей, быстрый запрос актуальной цены.
  • Извлечение адресов через regex: без LLM, чтобы не терять миллисекунды на задаче, где достаточно строгого формата Solana-адреса.
  • Стриминг блокчейна: предварительная обработка миграций PumpFun/PumpSwap и хранение данных с TTL в PostgreSQL.
  • SaaS-оболочка: изолированные воркспейсы, раздельное хранение данных и логов, настройка каналов и типов токенов для каждого клиента.
